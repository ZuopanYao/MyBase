  • corner

    设置圆角

    Parameters

    1. radius: 半径，这是必须的
    2. masksToBounds: 是否将子层裁剪到父层边界，保持默认，可传 nil

    Example

    let view = UIView()
view.corner = (50.0, nil)
// view.corner = (50.0, true)

    Declaration

    Swift

    public var corner: (radius: CGFloat, masksToBounds: Bool?) { get set }

  • shadow

    2. offset: 偏移, 保持默认，可传 nil
