  • title

    设置标题

    Parameters

    1. normal: 正常状态下的标题, 这是必须的
    2. highlighted: 高亮状态下的标题，保持默认，可传 nil
    3. selected: 选择状态下的标题，保持默认，可传 nil
    4. disabled: 禁用状态下的标题，保持默认，可传 nil

    Example

    let btn = UIButton()
btn.title = ("正常标题", "高亮时的标题", "选中状态标题", "禁用时标题")
